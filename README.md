# agente-ia-atendimento
Agente de IA de atendimento operacional, técnico e administrativos, com foco em redução de custo de estrutura, com foco em humanização. Funcionalidades: Atendimento via chat (API), regras de negócio por cliente, base de conhecimento (biblioteca com manual técnico), Detecção de contexto (ID de cliente). Phyton, FastAPI, Hugging LLMs, IA conversacion
